    Вызывать пересчет данных после обновления настроек профиля можно в методе onStop() или
onDestroyView() фрагмента.


Какие поля нужны в Норме?

- количество отработанных часов / планируемых часов
- количество выходных / планируемых выходных
- количество отработанных смен / количество планируемых отработанных смен
- количество ночей
- количество праздничных смен
- количество праздничных часов
